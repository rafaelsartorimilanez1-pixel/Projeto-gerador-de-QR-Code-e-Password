# 🔧 Projeto - Gerador de QR Code e Senhas

Este é um projeto simples em Node.js que funciona via terminal (CLI), permitindo ao usuário gerar **QR Codes** e **senhas seguras** de forma interativa.

---

## 🚀 Funcionalidades

- 📌 Menu interativo no terminal
- 🔳 Geração de QR Code
- 🔐 Geração de senhas seguras

---

## 📦 Tecnologias utilizadas

- Node.js
- Prompt (entrada de dados no terminal)
- JavaScript ES Modules
